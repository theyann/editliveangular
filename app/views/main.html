EditLive with AngularJS

This sample site is a playground to help to get EditLive to work with AngularJS.

The goal: we need EditLive to work with AngularJS the same way TinyMCE does. Checkout the links below, the first page uses tiny mce in a directive, the second uses EditLive in a directive. Both examples should work the same way.

Note: This project uses the ui-TinyMCE project to get TinyMCE to work with Angular. A similar approach could also work for EditLive.
